# Inventory-Management-System
A Java Swing-based Inventory Management System that allows users to manage product records through a graphical user interface.

This application allows adding, updating, and deleting products while displaying them in a dynamic table.

 Features

- Add new products with:
  - Product ID
  - Name
  - Quantity
  - Price
  - Created date
  - Updated date
- Update existing products
- Delete products
- Display all products in a table
- GUI built using Java Swing
- Input validation for unique Product IDs

 Concepts Demonstrated

- Java Swing (GUI Programming)
- Object-Oriented Programming (OOP)
- Event Handling (ActionListeners)
- Table Models (`DefaultTableModel`)
- Lists and Data Management
- MVC-like structure
